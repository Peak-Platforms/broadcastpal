1Day before — register the game

Open Supabase → Table Editor → games table → find the school's row → update the opponent name and game date.

Update via API (faster)

Or use the API directly from any terminal. Replace the game ID and details:

curl -X PATCH https://broadcastpal-server-production.up.railway.app/games/GAME_ID/status \
  -H "Content-Type: application/json" \
  -d '{"status":"upcoming"}'

2Game day — pre-game checklist

  • Confirm all 4 Railway services are online at broadcastpal-server-production.up.railway.app/health
  • Set game status to live in Supabase → games table → status column
  • Confirm glasses are charged and paired to MentraOS app on phone
  • Open director dashboard at broadcastpal.fun/director on your laptop
  • Share viewer link with fans: broadcastpal.fun/live
  • Test radio stream is live at stream.xsen.fun/listen/xsen_sooners/radio.mp3

Set game to live

In Supabase → Table Editor → games → find the game row → change status from upcoming to live. This makes it appear prominently in the viewer portal.

3Starting a stream — glasses operator

Each person wearing Mentra Live glasses follows these steps:

Glasses operator instructions
  1. Put on Mentra Live glasses
  2. Open MentraOS app on phone — confirm glasses are connected
  3. Open BroadcastPal from the MiniApp list
  4. Glasses display shows: 📡 BroadcastPal — Say "go live" to start streaming
  5. Say "go live" clearly
  6. Glasses display changes to: 🔴 LIVE — Stream key: xxxxxxxx
  7. You are now broadcasting your POV
To stop: Say "stop" or long-press the glasses button.

4Director — starting multiview

Once glasses operators are live, open broadcastpal.fun/director on your laptop.

Starting a multiview broadcast
  1. Active streams appear in the left panel as cards
  2. Tap each stream you want to include (up to 4)
  3. Select the score overlay team from the dropdown
  4. Click Start Multiview
  5. The composed stream is now available at:
    broadcastpal-nginx-production.up.railway.app/hls/composite/output.m3u8
  6. Fans watching at broadcastpal.fun/live see the Multiview option appear automatically

Switching audio mid-broadcast

In the director dashboard → find the running session → tap a Feed 1 / Feed 2 / Feed 3 / Feed 4 button under Audio Source. The live mic switches instantly — useful when different fans are in better positions at different points in the game.

Swapping the score overlay

Use the score overlay dropdown in the running session to switch between OU, OSU, and Texas without restarting the broadcast. The new overlay burns into the stream within a few seconds.

5What fans see

broadcastpal.fun/live

Standalone viewer portal. Share this link on social media, in group chats, or embed on your booster club site. Shows all active games, POV feeds, and the multiview. Includes radio and live score.

xsen.fun app.html

Full fan experience — Boomer Bot, radio, video highlights, and the live POV stream in one place. For existing XSEN subscribers. Live Video tab appears automatically when a stream is active.

6Post-game shutdown

  • Glasses operators say "stop" to end their streams
  • In director dashboard → click Stop on any running multiview sessions
  • In Supabase → games table → change game status from live to ended
  • Confirm viewer portal shows "Upcoming" or no active feeds

?Troubleshooting

ProblemFix
Glasses say "stream failed to start" Check WiFi signal. Move closer to access point. Try saying "go live" again. Check Railway services are online.
Stream doesn't appear in director dashboard Refresh the dashboard. Active streams update every 15 seconds. Confirm glasses display shows 🔴 LIVE.
Viewer portal shows 0 feeds Confirm game status is set to "live" in Supabase. Refresh portal — it updates every 30 seconds.
Video player shows black screen Multiview takes 10–15 seconds to start compositing after you click Start Multiview. Wait and refresh.
Audio sounds wrong or silent Use the Audio Source switcher in the director dashboard to select a different feed.
Score overlay not updating Overlay refreshes every 45 seconds from ESPN. If no game is found, overlay won't show. Confirm ESPN team ID matches the game being played.
Railway services offline Go to railway.app → BroadcastPal Core project → restart any failed services.
