Larix RTMP Test Guide

Simulate a glasses stream using your phone · BroadcastPal · Peak Platforms

Larix Broadcaster is a free iOS/Android app that pushes RTMP video from your phone camera — same protocol the Mentra Live glasses use. Use this to test the full BroadcastPal chain before your glasses arrive.

What this proves: Phone camera → nginx RTMP ingest → FFmpeg compositor → HLS output → fan viewer. Every link in the chain. The only difference with real glasses is the source device.

1Install Larix

  • iOSSearch "Larix Broadcaster" in the App Store — free, by Softvelum
  • AndroidSearch "Larix Broadcaster" in Google Play — same app

2Configure the RTMP connection

In Larix: tap the Settings gear → Connections → + to add a new connection.

Name
BroadcastPal Test
URL
rtmp://shortline.proxy.rlwy.net:12083/live
Stream key
test1234
Stream key = section label. Use any short identifier — test1234, section214, endzone. This becomes the feed name visible in the director dashboard and viewer portal. For multiple simultaneous test streams use different keys per device.

3Register the stream against a game

Before hitting go in Larix, register your test stream in the game registry so it appears in the viewer portal. Run this in PowerShell:

Invoke-WebRequest `
  -Uri "https://broadcastpal-server-production.up.railway.app/games/7a8d9693-3e31-43d6-94bb-cc8d537b8a82/streams" `
  -Method POST `
  -ContentType "application/json" `
  -Body '{"streamKey":"test1234","userId":"test@test.com","label":"Test Feed","section":"Test"}' `
  -UseBasicParsing
Game IDs: OU = 7a8d9693-3e31-43d6-94bb-cc8d537b8a82 · OSU = f7df2de2-37fd-49bf-8ed9-b91b23730ed6 · Texas = 03f1d73b-b94c-4df3-be58-1b15dc140a07

4Start streaming in Larix

  • 1Point your phone camera at something — doesn't matter what
  • 2Tap the red record button in Larix
  • 3Larix shows Connected and a live timer — stream is flowing

5Verify the full chain

CheckURL / ActionExpected
Stream registered /streams {"active":[...],"count":1}
Viewer portal broadcastpal.fun/live OU card shows 1 feed
Individual HLS /hls/test1234/index.m3u8 Opens in VLC — live video
Director dashboard broadcastpal.fun/director Test Feed card appears

6Start multiview from director

  • 1Open broadcastpal.fun/director
  • 2Tap the Test Feed card to select it
  • 3Select Oklahoma Sooners as score overlay
  • 4Click Start Multiview
  • 5In viewer portal — click OU game → Multiview option appears
  • 6Video plays — phone camera feed composited with score overlay
Full chain confirmed when you can watch your phone camera feed playing at broadcastpal.fun/live with a score overlay. That is identical to what Mentra Live glasses will produce — different source device, same pipeline.

7Testing multiple feeds

To simulate multiple glasses at once — have a second phone run Larix with a different stream key (e.g. test5678), register it against the same game, and start multiview with both feeds selected. The director dashboard will show a 2-up layout.

8Cleanup after testing

  • 1Stop streaming in Larix — tap the red button again
  • 2Stop multiview in director dashboard
  • 3Delete test stream from Supabase → streams table
